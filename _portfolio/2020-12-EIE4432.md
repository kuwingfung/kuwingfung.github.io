Milestone project for the Web Systems and Technologies course at PolyU. An online examination system site to mimic the existing learning management system such as Blackboard, Moodle, and Canvas. In this project, we used Apache, PHP and MySQL as web application infrastructure to construct the site, with the assist of javascript and HTML. JQuery and Bootstrap are also used in the frontend to enhance user interface and user experience.
