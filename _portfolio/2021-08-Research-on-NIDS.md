Traditonal methods has to rely on feature engineering. This often requires domain knowledge, and its time consuming to extract the features from millions of packet data. 

Our work uses deep learning to achieve End-to-End Network Intrusion Detection System. Our method detects Network Intrusion only with few bytes. This is done by turning binary information into image and feed into a vision model to extract the features then feed to an autoencoder model.
