PolyU Robotics Team Projects

I joined my university robotics team since year 1. (2019 Jan)

ABU Robocon 2019 (I was Year 1)

Projects/Contribution I did for Robotics Team:

  • Color Object Detection program (Color tracking)
  • Human face Detection program (Human face using haar cascade and dlib)
  • Microcontroller program to control motor movement for shooting mechanism of a robot

The Microcontroller program was selected and featured in the automatic robot during ABU Robocon 2019 (HK).

ABU Robocon 2019 Result:

  • Our team scored 4th place in ABU Robocon 2019 (HK)
  • Our Team (Crimson) Won the Best Engineering Award

ABU Robocon 2020 (I was Year 2)

Due to the Siege of the Hong Kong Polytechnic University and COVID-19 pandemic in Hong Kong, school was closed for a long time and we suffer from resources shortage. We were mainly learning new skills and I spent my most of the time taught myself deep learning.

Projects/Contribution I did for Robotics Team:

  • Started using Linux Ubuntu for development (2019 Sep), taught myself Linux CLI and basic level of ROS (Robot Operating System)
  • Object Detection program (Background Subtraction) using depth camera, integrated on ROS
  • Taught myself CNN (Convolutional neural network) algorithm and started using YOLO object detection
  • Collaborated with PolyU ROV team and hosted a public workshop “Brief Introduction on Computer Vision”

The Color tracking program I did in year 1 was featured in a workshop “Brief Introduction on Computer Vision” opened to PolyU students where I was the main presenter. (2020 Dec)
